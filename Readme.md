# mdbp-example

  mobile first markup driven build example

## Installation

  Clone this repo

    $ git clone https://github.com/dombojs/mdbp-example.git

  Install the global library with [component(1)](http://component.io):

    $ component install

  Initialize mdbp

    $ mdbp init mdbp-mobile-first-dombo

## Usage

  Scan application

    $ mdbp scan

  Run build process

    $ mdbp run
